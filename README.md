# 项目简介

## 核心功能
- **登录注册**：用户可通过小程序完成账号注册与登录
- **非遗学习内容**：提供课程学习与VR体验功能
- **研学线路浏览与报名**：支持研学线路查看和预约报名
- **学习记录**：记录用户的学习进度和历史
- **个人中心**：管理个人信息和功能入口
- **内容管理**：后台支持学习内容的发布与编辑
- **研学管理**：后台负责研学线路的设置与管理
- **报名管理**：后台处理研学报名相关事务
- **统一API**：后端提供小程序与后台管理的数据接口
- **鉴权服务**：保障系统安全访问
- **数据与文件服务**：处理系统数据存储和文件管理

## 适用场景
- 非物质文化遗产的学习与传播
- 研学活动的组织与参与
- VR技术在文化教育领域的应用
- 学术论文研究与资料收集
- 360全景技术在文化展示中的应用

## 技术特点
- 采用Spring Boot构建后端服务，提供稳定的数据处理能力
- 前端使用uni-app开发，实现跨平台小程序体验
- Vue3技术栈构建管理后台，提供直观的操作界面
- 集成VR与360全景技术，增强学习体验的沉浸感
- 模块化设计，便于功能扩展与维护
